使用說明 📘

五分鐘看懂這個網站怎麼用，以及怎麼把它的價值最大化。

這個網站是什麼？

一本為產品經理（PM）設計的 Claude 入門手冊。架構設計參考了教育領域的 Claude 教學手冊的學習方法（概念 → 引導 → 實作 → 資源），內容全部針對 PM 的工作情境重新編寫： 需求規格、用戶研究、數據分析、利害關係人溝通、競品分析與規劃排序。

你不需要任何技術背景。需要準備的只有：一個 Claude 帳號（claude.ai，滿 18 歲）、以及你手上真實的工作任務。

四大區塊怎麼用？

區塊用途建議用法
📖 概念學習建立心智模型依序讀八章，每章結尾的「自我檢核」通過再往下。約 60 分鐘。
🧭 引導式學習動手前的校準每個工具 3～5 分鐘。「模式診斷」與「Prompt 組裝器」在每次開始新任務前都可以重複使用。
🛠️ 任務實作真實任務練習用篩選器（領域／難度／模式）找到適合的任務，展開卡片照步驟做。從「入門」開始。
📚 協作資源日常查用加入書籤。Prompt 資料庫與詞彙表是使用頻率最高的兩頁。

互動功能操作說明

1. 複製 Prompt

所有深色的 Prompt 區塊右上角都有「複製」按鈕，點擊即複製全文到剪貼簿，直接貼到 Claude 對話框，把【】內的占位文字替換成你的內容即可。

2. 任務篩選器

在「任務實作」頁，點選工作領域、難度、模式的篩選按鈕，任務清單會即時過濾；點擊任務卡的標題列可展開／收合完整內容。

3. 互動工具

  • 模式診斷：三題都作答後按「診斷結果」。
  • Prompt 組裝器：下拉選單選擇或自行填寫，按「組合 Prompt」後即可一鍵複製。
  • 4D 自評：拖動滑桿打分，按「看評估結果」取得最弱維度與改善建議。
  • Agentic 模擬器：連續按「下一步」觀察四個階段，注意兩個「PM 介入點」。

4. 學習進度建議

本站不儲存任何個人資料，也沒有帳號系統。建議直接把有用的頁面存書籤、把驗證過的 Prompt 存進你自己或團隊的筆記工具。

常見問題

Q：一定要照順序學嗎？

不一定。趕時間的話：讀第 3、4 章 → 模式診斷 → 做一個入門任務，30 分鐘就能開始產生價值。但第 8 章（負責任的使用）請務必在把公司資料貼進 AI 之前讀完。

Q：內容適用其他 AI 工具嗎？

心智模型（模式選擇、四元素 Prompt、4D 框架、治理原則）是通用的；操作細節以 Claude 為準。

Q：發現內容過時或有錯？

AI 產品迭代很快，功能細節請以 官方資源 為準。本站原始碼在 GitHub 上，歡迎提交 issue。
